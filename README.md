# 🤖 Dify智能客服机器人 

## 📌 项目简介
这是一个基于Dify云平台和DeepSeek大模型搭建的智能客服助手。能够根据产品FAQ文档，自动回答用户咨询，支持多轮对话和转人工策略。

## 🚀 在线体验
👉 [点击此处与客服机器人对话](这里粘贴你的Dify应用分享链接)

*（无需登录，打开即用）*

## 🛠️ 技术栈
- **平台**：Dify（开源LLM应用开发平台）
- **模型**：DeepSeek Chat（通过OpenAI-API-compatible接入）
- **知识库**：基于产品FAQ文档的RAG检索增强生成
- **部署**：Dify Cloud（零代码，纯拖拽配置）

## 📁 项目内容
- `/docs` - 知识库原始文档（产品FAQ）
- `/screenshots` - 项目配置和对话效果截图
- `prompt.txt` - 客服助手的提示词配置

## ✨ 核心功能
- ✅ 基于产品文档的自动问答
- ✅ 多轮对话上下文理解
- ✅ 知识库检索增强生成（RAG）
- ✅ 友好的人设话术（开场白、安抚话术）
- ✅ 超出知识范围时引导转人工

## 🧠 项目亮点
- **2天快速落地**：从零到可访问的客服机器人，验证了RAG在客服场景的有效性
- **真实业务模拟**：基于真实产品FAQ构建知识库，贴近工业落地
- **可扩展性强**：可无缝升级到Agent模式，对接订单查询等工具调用
